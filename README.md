# Compagnon Étudiant

Une application web simple pour aider les étudiants à organiser leurs devoirs, voir les cours annulés et accéder à des ressources d'apprentissage.

## Fonctionnalités principales
- Voir la liste des devoirs
- Consulter les cours annulés
- Accéder à des notes/cours

## Stack technique
- Frontend : React (en français)
- Backend : Node.js + Express
- Stockage : JSON local

## Démarrage rapide
1. `npm install` dans les dossiers `frontend` et `backend`
2. `npm start` dans chaque dossier
3. Accédez à l'application sur http://localhost:3000
