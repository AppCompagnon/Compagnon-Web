# Compagnon

Une application web simple pour aider les étudiants à organiser leurs devoirs, voir les cours annulés et accéder à des ressources d'apprentissage.

Compagnon est open source, signifiant que son code est public, pour nous, il est disponible sur GitHub. Mettre notre code en open source est un moyen de prouver l'authenticité et la sécurité de l'application.
